## 技术架构
PHP + ThinkPHP6 + LayUI + MySQL

## 安装
```zsh
composer install
```

安装Think模版
```zsh
composer require topthink/think-view
```
安装验证码模块
[文档](https://www.kancloud.cn/manual/thinkphp6_0/1246402)
```zsh
composer require topthink/think-captcha
```

## 要求

ThinkPHP的版本是`6.0.13`

PHP >= `7.4`
